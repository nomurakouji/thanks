## 開発言語
- ruby 3.0.1
- rails 6.0.4.4

## 就業Termの技術
- devise
- お気に入り機能

## カリキュラム外の技術
ransack

## 実行手順
```
$ git clone　git@github.com:nomurakouji/thanks.git
$ cd thanks
$ bundle install
$ rails db:create 
$ rails db:migrate
$ rails s
```
